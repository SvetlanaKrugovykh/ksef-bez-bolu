# KSeF bez bólu

A simple, modern multilingual feedback/contact landing page for KSeF support.

## How to deploy on GitHub Pages

1. Rename `kse_f_bez_bolu_draft_site.html` to `index.html`.
2. Push `index.html` and `modals.js` to your GitHub repository (e.g., `yourname/ksef-bez-bolu`).
3. In repository settings, enable GitHub Pages from the root (`/`) of the main branch.
4. Your site will be available at `https://yourname.github.io/ksef-bez-bolu/`.
